使用指南 Usage Guidelines

  • 可访问性: 所有文本和重要元素保持 WCAG AA 对比度标准
  • 色盲友好: 橙色和蓝色组合为大多数色觉类型提供良好的可见性
  • 环境适应: 浅色背景适合明亮环境，默认方案适合大多数照明条件
  • 强调色: 橙色为重要元素提供强烈的强调色
  • 实现方式: 使用 CSS 自定义属性或 JSON 配置进行程序化访问
